# Contributing to DB Migration Architect Agent

Thank you for considering contributing to the DB Migration Architect Agent! We welcome contributions from the community.

## How to Contribute

1. Fork the repository
2. Create a new branch for your feature or bug fix
3. Make your changes
4. Ensure your code follows our coding standards
5. Add tests for any new functionality
6. Submit a pull request

## Coding Standards

- We use TypeScript for all backend code
- Follow the ESLint and Prettier configurations in the repository
- Write clear, descriptive commit messages
- Add JSDoc comments for all public APIs
- Keep functions small and focused

## Development Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables: copy `.env.example` to `.env` and fill in the values
4. Run the build: `npm run build`
5. Run tests: `npm test`

## Reporting Issues

Please use the GitHub issue tracker to report bugs or request features. When reporting a bug, please include:
- Steps to reproduce the issue
- Expected behavior
- Actual behavior
- Any relevant logs or error messages
